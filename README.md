# core-engine
================

## Description
------------

The core-engine is an open-source, high-performance game engine designed for building complex game applications. It provides a robust set of features, including physics, graphics, animation, and AI, to create immersive gaming experiences.

## Features
------------

### Core Features

* **Physics Engine**: A 3D physics engine that supports rigid body dynamics, collision detection, and response.
* **Graphics Engine**: A high-performance graphics engine that supports DirectX 12, OpenGL 4.5, and Vulkan 1.2.
* **Animation System**: A state machine-based animation system that supports keyframe animation, blend trees, and physics-based animation.
* **AI System**: A behavior tree-based AI system that supports pathfinding, navigation, and decision-making.

### Advanced Features

* **Multi-Threading**: The core-engine supports multi-threading, allowing for better performance on multi-core processors.
* **Memory Management**: A custom memory management system that reduces memory leaks and improves performance.
* **Diagnostics**: A comprehensive diagnostics system that provides real-time performance metrics and error reporting.

## Technologies Used
--------------------

* **C++**: The core-engine is built using C++11, providing a high level of performance and flexibility.
* **OpenGL**: The graphics engine uses OpenGL 4.5 for rendering and graphics processing.
* **DirectX**: The graphics engine also supports DirectX 12 for rendering and graphics processing.
* **Vulkan**: The graphics engine uses Vulkan 1.2 for rendering and graphics processing on Windows and Linux platforms.
* **Boost**: The project uses Boost libraries for various tasks, including file I/O, networking, and thread management.

## Installation
--------------

To install the core-engine, follow these steps:

### Prerequisites

* **C++11 Compiler**: Install a C++11 compiler, such as GCC 5.1 or later.
* **OpenGL**: Install the OpenGL SDK.
* **DirectX**: Install the DirectX SDK.
* **Vulkan**: Install the Vulkan SDK.
* **Boost**: Install the Boost libraries.

### Installation Steps

1. Clone the repository using the following command:
   ```bash
git clone https://github.com/core-engine/core-engine.git
```
2. Navigate to the project directory:
   ```bash
cd core-engine
```
3. Build the project using the following command:
   ```bash
cmake .
make
```
4. Run the project using the following command:
   ```bash
./core-engine
```
